# First Steps with QuantomDocs

Welcome to QuantomDocs! This guide will help you navigate and make the most of our documentation platform.

## What is QuantomDocs?

QuantomDocs is a comprehensive documentation portal designed to help you get the most out of Quantom Server. Whether you're a beginner setting up your first server or an experienced administrator optimizing performance, you'll find everything you need here.

## Navigation Overview

### Header Navigation

At the top of every page, you'll find:

- **Home** - Return to the main landing page
- **Documentation** - Access all documentation (you are here!)
- **Downloads** - Download the latest Quantom Server versions
- **Discord** - Join our community for support and discussion
- **Theme Toggle** - Switch between dark and light modes

### Product Navigation

Below the main header on documentation pages, you'll see product tabs. Currently, we focus on:

- **Quantom Server** - Our flagship Minecraft server software

As we grow, additional products will appear here.

### Documentation Sidebar

The left sidebar shows all documentation categories for the current product:

- **Getting Started** - Installation guides and quick start tutorials
- **Configuration** - Server configuration and settings
- **Guides** - Step-by-step tutorials for common tasks
- **Reference** - Technical reference documentation
- **Tools** - Utility tools and helper scripts

### Table of Contents

The right sidebar (desktop only) displays the current page's table of contents. Click any heading to jump directly to that section.

## Search Functionality

Use the powerful search feature to find what you need quickly:

1. Click the search icon or press `Ctrl+K` (Windows/Linux) or `Cmd+K` (Mac)
2. Type your search query - typos are okay, our fuzzy search will find what you need!
3. Use arrow keys to navigate results
4. Press Enter to open the selected result

### Search Features

- **Fuzzy Matching** - Finds results even with typos
- **Category Filters** - Filter results by documentation category
- **Search History** - Quick access to recent searches
- **Relevance Badges** - See how well each result matches your query

## Page Actions

On documentation pages, you'll find helpful actions in the top-right corner:

### Copy Page

Click the main button to instantly copy the page content as Markdown - perfect for sharing with AI assistants or colleagues.

### Additional Actions

Click the dropdown arrow for more options:

- **Copy Page** - Same as the main button
- **View as Markdown** - See the raw Markdown source
- **Download Page** - Save the page as a `.md` file

## Documentation Features

### Code Blocks

All code examples include:

- **Syntax Highlighting** - Easy-to-read color-coded syntax
- **Language Labels** - Clear indication of the programming language
- **Copy Button** - One-click copying with visual feedback

Example:

```yaml
# Server configuration example
server:
  port: 25565
  max-players: 100
  motd: "Welcome to Quantom Server!"
```

### Breadcrumbs

At the top of each page, breadcrumbs show your current location:

```
Home / Documentation / Quantom Server / Getting Started / First Steps
```

Click any breadcrumb to navigate back to that level.

### Active Section Highlighting

As you scroll through a page, the table of contents automatically highlights the section you're currently viewing.

## Offline Access

QuantomDocs works offline! After your first visit:

- All pages are cached for offline viewing
- Search functionality remains available
- Automatic updates when new content is available

When you're offline, you'll see a notification at the top of the page.

## Responsive Design

QuantomDocs adapts to your device:

- **Desktop** - Full three-column layout with sidebars
- **Tablet** - Optimized two-column layout
- **Mobile** - Streamlined single-column view with hamburger menu

| Shortcut | Action |

Switch between dark and light themes using the theme toggle button in the header. Your preference is saved automatically.

- **Dark Mode** - Easier on the eyes in low-light environments
- **Light Mode** - Better for bright environments or printing
